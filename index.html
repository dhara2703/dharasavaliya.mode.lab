Mobile Application Development Tool


Tool Name & Version

Xamarin - Latest released version: Xamarin.Forms 4.2


Platform

Having Cross-platform and open-source features, Xamarin extends the .NET developer platform with tools and libraries specifically for building apps for Android, iOS, tvOS, watchOS, macOS, and Windows.

Xamarin separates the concerns of presentation, presentation logic, and entities through support for the Model-View-ViewModel (MVVM) pattern.


Application Development Type

  • Type: Cross-Platform
  • Advantages:
    1. Reusable Code Components
    2. Speed
    3. Reduced Costs
  • Disadvantages:
    1. Performance Challenges
    2. Slow Code Performance With Limited Tool Availability
    3. Limited User Experience
